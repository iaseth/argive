
# argive
[`argive`](https://www.npmjs.com/package/argive) is a library for parsing command line arguments.



## Installation
You can install [`argive`](https://www.npmjs.com/package/argive) with the following command:
```
npm i argive
```

Now you should be able to import `argive` in your apps.
```
import argive from "argive";
```
